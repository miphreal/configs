# Intro

A bundle of user configs

There are conky, shell, vim, todo, gnome-shell, upstart jobs.

# How to install:

    $ git clone git@github.com:miphreal/configs.git
    $ cd configs
    $ . install-shell
    $ . install-vim
    $ . install-conky
    $ . install-todo
    $ . install-gnome-shell-ext
    $ . install-upstart
